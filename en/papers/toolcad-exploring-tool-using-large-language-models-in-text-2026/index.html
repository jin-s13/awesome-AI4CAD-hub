Deep Analysis

Innovations

  • ToolCAD: an agentic CAD framework deploying LLMs as tool-using agents for text-to-CAD generation
  • Interactive CAD modeling gym for rolling out reasoning and tool-augmented interaction trajectories with hybrid feedback and human supervision
  • End-to-end post-training strategy with CAD Modeling Chain of Thought (CAD-CoT) and online curriculum reinforcement learning to refine LLM agents

Methodology

ToolCAD uses LLMs as tool-using agents that interact with a CAD engine via an interactive gym environment, generating trajectories with hybrid feedback and human supervision. The LLM is post-trained end-to-end using online curriculum reinforcement learning to improve CAD-specific chain-of-thought reasoning and tool use.

Key Results

ToolCAD enables open-source LLMs to perform comparably to proprietary models on text-to-CAD generation, demonstrating effective training of CAD tool-using agents.

深度分析

创新点

  • 提出知识充分性作为示例选择的原则性目标，以满足设计规范中的所有需求。
  • 引入设计规范平铺（DST），通过使用多粒度设计组件和覆盖度量，利用平铺比量化知识充分性。
  • 将示例选择形式化为子模最大化，并通过贪心算法提供(1-1/e)近似保证。

方法

DST从CAD设计规范中提取多粒度设计组件，并选择示例以最大化查询组件被覆盖的比例（平铺比）。该选择被形式化为子模最大化，并通过多项式时间贪心算法求解。实验将DST与ICL中现有的示例选择策略在CAD代码生成上进行了比较。

关键结果

DST显著提高了CAD代码生成质量，并在ICL中持续优于现有的示例选择策略。

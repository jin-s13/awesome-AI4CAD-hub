Deep Analysis

Innovations

  • B-Rep Gaussian Splatting (BrepGaussian) framework that learns 3D parametric representations from 2D images
  • Two-stage learning framework: first captures geometry and edges, then refines patch features for clean geometry and coherent instance representations
  • Use of Gaussian Splatting renderer with learnable features and a specific fitting strategy

Methodology

The method employs a Gaussian Splatting renderer with learnable features and a two-stage learning framework. The first stage captures geometry and edges, while the second stage refines patch features to produce clean geometry and coherent instance representations from multi-view 2D images.

Key Results

Extensive experiments demonstrate superior performance compared to state-of-the-art methods.

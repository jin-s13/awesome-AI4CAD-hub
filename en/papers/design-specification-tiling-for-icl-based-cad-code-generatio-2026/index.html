Deep Analysis

Innovations

  • Proposes knowledge sufficiency as a principled objective for exemplar selection to satisfy all requirements in design specifications.
  • Introduces Design-Specification Tiling (DST) that quantifies knowledge sufficiency via a tiling ratio using multi-granular design components and coverage measurement.
  • Formulates exemplar selection as submodular maximization with a greedy algorithm providing (1-1/e)-approximation guarantee.

Methodology

DST extracts multi-granular design components from CAD design specifications and selects exemplars to maximize the proportion of query components covered (tiling ratio). The selection is formulated as submodular maximization and solved with a polynomial-time greedy algorithm. Experiments compare DST against existing exemplar selection strategies in ICL for CAD code generation.

Key Results

DST substantially improves CAD code generation quality and consistently outperforms existing exemplar selection strategies in ICL.

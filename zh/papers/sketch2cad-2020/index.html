Sketch2CAD

arXiv 2020 49.5 method, system, application

TLDR

一个基于深度学习的草图CAD系统，通过合成数据训练将连续草图解释为CAD操作。

评分理由

The paper presents a novel approach integrating sketch input with CAD modeling via deep learning, supported by a user study. However, it only supports four CAD operations and relies on synthetic training data, limiting generalizability.

深度分析

创新点

  • 利用先前步骤构建的部分CAD模型作为上下文进行增量式草图解释
  • 联合识别目标CAD操作并对输入草图进行分割的深度神经网络
  • 通过将CAD操作序列渲染为线条图来生成合成训练数据
  • 基于优化的参数估计方法，从分割后的草图笔画中估计操作参数以完成CAD操作

方法

该系统接收用户的草图以及先前步骤中的部分CAD模型，将其输入深度神经网络，该网络预测CAD操作并对草图笔画进行分割，随后通过优化估计最匹配分割笔画的操作参数。训练依赖于渲染为线条图的合成CAD操作序列。

关键结果

在用户研究中，参与者使用概念验证系统快速建模了大量多样化的物体，证明了Sketch2CAD作为CAD建模的一种可行替代交互方法。

局限性

  • 概念验证仅支持四种CAD操作
  • 在合成线条图上训练，可能无法完全捕捉真实用户草图的多样性
  • 摘要中未报告定量性能指标

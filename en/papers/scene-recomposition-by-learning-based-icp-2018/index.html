Deep Analysis

Innovations

  • Fully automatic scene recomposition system that matches, places, and aligns objects in a room scan to a database of CAD models.
  • Learning-based ICP (LICP): a deep reinforcement learning approach for aligning CAD models to 3D scans, learning optimal points to match and conditioning on viewpoint.
  • LICP trained solely on synthetic data without requiring real scene pose annotations or keypoint pair matching, yet outperforms both learned local feature matching and geometric alignment methods on real scenes.

Methodology

A depth sensor is moved around a room to capture a 3D scan; objects are matched to a CAD model database and aligned using Learning-based ICP, which employs deep reinforcement learning to select points for matching and conditions on object viewpoint. The model is trained exclusively on synthetic data.

Key Results

LICP outperforms prior ICP methods, learned local deep feature matching, and geometric alignment methods on real scene datasets (SceneNN, ScanNet) and synthetic scenes (SUNCG), showing robustness to clutter, viewpoint, and occlusion.
